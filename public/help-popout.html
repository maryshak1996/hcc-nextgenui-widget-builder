Help window (optional page)

Undocked Help from the console opens a dedicated about:blank window and injects the Help panel from the main app (it does not load this HTML file). This page exists for manual testing or documentation only.

Opened this URL in a normal tab? This HTML file has no JavaScript bundle. To load real Help content, use Help → Open Help in a separate window from the Hybrid Cloud Console.

If you already used undock from the console and still see only this message, try closing this tab and opening Help in a separate window again.

Reloaded this tab? A full refresh drops the link to the console; close this tab and use Help → Open Help in a separate window again from the main app.
